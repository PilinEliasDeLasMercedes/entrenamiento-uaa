🎓 EXANI-II Master - Entrenador IA para la UAA

Este es un simulador de examen de admisión adaptativo diseñado específicamente para aspirantes a la Universidad Autónoma de Aguascalientes (UAA). Utiliza la IA de Google Gemini para generar ejercicios únicos basados en el temario del EXANI-II.

🚀 Características Útiles

Dificultad Adaptativa: La IA analiza tus respuestas y sube el nivel (1-10) conforme mejoras, hasta igualar la complejidad real del examen UAA.

Progresión Estricta: Sistema de desbloqueo por días y semanas para asegurar una preparación constante.

Radar de Habilidades: Visualiza tus fortalezas en Física, Cálculo, Aritmética, Redacción y Lógica.

Privacidad: Tu progreso se guarda localmente en tu navegador.
